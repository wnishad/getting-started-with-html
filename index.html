What is HTML?

html stands for hypertext markup language, which is a standardized system for font, color, graphic, and other effects on web pages.
HTML is made up of
  -text content
  -markup (aesthetic)
  -references to (images, media, etc.)
  -links

HTML basics

What is a tag?
A tag tells the computer how to format and display the information in the way you want it to.
Tags start with an opening tag and end with a closing tag, telling the computer what exactly should be formatted in what way. For example, I have formatted these notes with header tags to make the title and headers bigger and bold.
I can also add images using tags, like this: #freeguwop
An image tag is an example of a tag that doesn't need a closing tag because it's not modifying a pre-existing element. Rather, it is adding a new one.

What is an attribute?
An attribute modifies elements in HTML to suit a specific purpose. A link is a type of a attribute.
